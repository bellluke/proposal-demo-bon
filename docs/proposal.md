# 프로젝트 제안서

## 파충류 전문 커뮤니티 통합 커머스 플랫폼 구축

**수신:** 주식회사 비오엔(B.O.N) 김재웅 대표님
**발신:** (주)에이나
**일자:** 2025년 00월 00일

---

## 1. 프로젝트 이해

비오엔의 핵심 브랜드 '두렙(DooRep)'은 국내 파충류 용품 시장에서 독자적인 기술력과 브랜드 철학을 갖춘 선도 브랜드입니다. 본 프로젝트는 단순 판매 중심의 쇼핑몰을 넘어, **유저 커뮤니티와 커머스가 유기적으로 결합된 '파충류판 오늘의집'**을 구축하는 것을 목표로 합니다.

이를 위해 다음 세 가지를 핵심 가치로 설정하고 플랫폼을 설계합니다.

- **콘텐츠가 곧 구매로**: 사육장 자랑하기 콘텐츠 내 상품 태깅을 통해 자연스러운 구매 전환 유도
- **재방문과 충성도**: 출석체크, 룰렛, 등급제 등 게이미피케이션 요소로 일일 방문 습관 형성
- **브랜드 정체성 구현**: 비오엔의 '생명과 자연의 가치'를 UI/UX 전반에 녹여낸 플랫폼 경험

---

## 2. 진행 전략: 2단계 런칭

안정적인 서비스 런칭과 품질 확보를 위해 **웹 우선 런칭 후 앱 확장**의 2단계 전략을 제안드립니다.

| 구분 | 1단계: 웹 런칭 | 2단계: 앱 런칭 |
|------|---------------|---------------|
| **범위** | 반응형 웹 (카페24 기반) | 하이브리드 앱 (Android + iOS) |
| **기간** | 9주 | 2주 |
| **목표** | 커뮤니티 커머스 플랫폼 완성 및 안정화 | 앱 스토어 출시 및 Push 알림 연동 |

**2단계 분리 사유**: 웹 플랫폼의 핵심 기능과 운영 안정성을 충분히 확보한 뒤 앱을 출시함으로써, 앱 스토어 심사 리스크를 최소화하고 초기 앱 사용자에게 완성도 높은 경험을 제공할 수 있습니다.

## 3. 구현 범위 및 접근 방식

### 3.1 쇼핑몰 기본 구조 (카페24 활용)

전체메뉴, 브랜드관, B급 할인관(특가), 카테고리(사료/영양제, 적재사육장, 은신처, 꾸미기용품, 렉사 등) 등 **쇼핑몰 기본 메뉴 구성은 카페24의 표준 기능을 최대한 활용**하여 구축합니다. 검증된 카페24 커머스 인프라를 기반으로 하므로 주문·결제·배송·재고 관리 등 핵심 커머스 기능의 안정성을 확보하면서 개발 효율을 높일 수 있습니다.

### 3.2 특화 페이지 기획 (레퍼런스 벤치마킹)

요청하신 레퍼런스 사이트를 면밀히 벤치마킹하여 기획을 진행합니다.

| 특화 페이지 | 벤치마킹 방향 |
|------------|-------------|
| **베스트 리뷰 모아보기** | 오늘의집의 매거진형 콘텐츠 큐레이션 방식을 참고하여, 고퀄리티 리뷰를 시각적으로 임팩트 있게 노출 |
| **MD 큐레이션** | 화방넷의 콘텐츠-상품 연동 구조를 참고하여, 상황별(질병, 청소, 입문 등) 맞춤형 제품 제안 페이지 구성 |
| **사육장 자랑하기** | 오늘의집의 유저 콘텐츠 피드 구조를 참고하되, 파충류 커뮤니티 특성에 맞게 피드형 게시판 설계 |
| **전체 UI 톤앤매너** | 랭킹닭컴의 커머스-커뮤니티 영역 분리 방식을 참고하여, 쇼핑과 콘텐츠가 자연스럽게 공존하는 레이아웃 설계 |

### 3.3 상품 태깅 시스템 (핵심 기능)

본 프로젝트의 차별화 핵심인 **상품 태깅 기능을 커뮤니티 콘텐츠 내에 유기적으로 구현**합니다.

- **사육장 자랑하기 피드**에 업로드된 사진 위에 상품 태그(+)를 좌표 기반으로 배치
- 태그 클릭 시 상품 정보 팝업 노출 → 상세 페이지 이동 → 즉시 구매까지 끊김 없는 흐름 구현
- 유저가 직접 콘텐츠를 올리고 상품을 태깅할 수 있는 에디터 제공
- 카페24 상품 데이터와 실시간 연동하여 가격·재고 정보 동기화

> 유저의 사육 환경 자랑이 곧 자연스러운 상품 추천이 되는 구조를 통해, 광고가 아닌 콘텐츠로서의 구매 전환을 실현합니다.

### 3.4 등급·포인트·쿠폰 통합 리워드 시스템

게이미피케이션, 멤버십 등급제, 리워드를 **하나의 통합 시스템**으로 설계하여 일관된 사용자 경험을 제공합니다.

**등급 시스템**
- 구매 금액/횟수 기반 자동 등급 산정 (실버 → 골드 → VIP → VVIP)
- 등급별 차등 혜택 자동 지급 (할인권, 배송비 무료 등)
- 유료 멤버십 가입 및 전용 혜택 제공

**포인트 & 쿠폰**
- 출석체크(개근 시 쿠폰), 룰렛(매일 포인트) 등 게이미피케이션 요소를 포인트 체계와 통합
- 등급 혜택 · 이벤트 참여 보상 · 구매 적립이 하나의 포인트로 수렴
- 향후 '도마뱀 육성 게임' 등 확장 콘텐츠와 연동 가능한 데이터 구조 사전 설계

### 3.5 하이브리드 앱 (2단계)

일정과 시스템 안정성을 고려하여 **앱 개발은 2단계에서 진행할 것을 제안**드립니다.

- 1단계에서 구축된 웹 플랫폼이 충분히 안정화된 후, 하이브리드 앱으로 래핑하여 출시
- **Push 알림** 구현 (쿠폰 만료, 특가 안내, 커뮤니티 알림 등)
- **카카오 알림톡** 연동
- Google Play Store / Apple App Store **심사 및 등록 대행 포함**

## 8. 사전 질문 답변

### Q1. 카페24 Admin API 활용 및 복잡한 로직 개발 경험

최근 **미즈노 코리아 공식 쇼핑몰을 카페24 기반으로 재구축**하는 프로젝트에 참여한 바 있습니다. 해당 프로젝트에서 쇼핑몰·오프라인 매장·ERP를 연계한 **통합 포인트 및 쿠폰 시스템**을 구현하였으며, 카페24 Admin API를 활용한 커스텀 로직 개발 경험을 보유하고 있습니다.

룰렛 및 관련 게이미피케이션 기능은 **별도 데모 페이지를 제작**하였으니 확인 부탁드립니다.

### Q2. 이미지 좌표 기반 상품 태깅 기능 구현 경험

상품 태깅 기능에 대해 **데모 페이지를 제작**하였습니다. 실제 동작하는 형태로 확인하실 수 있으니 검토 부탁드립니다.

### Q3. 앱 스토어 심사·출시 대행

당사 **회사소개서의 참여 프로젝트 이력**을 참고해 주시기 바랍니다. 대부분의 프로젝트가 웹과 앱을 함께 개발하는 프로젝트로, 하이브리드 앱 개발부터 Google Play Store / Apple App Store 심사 및 출시까지 전 과정을 대행합니다.

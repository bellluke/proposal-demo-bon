### 1. 하이잡하이유 – 서울시 특성화고 모바일 학생증

**블록체인·마이데이터 기반 청소년 교육이력관리 서비스 플랫폼**

| 항목 | 내용 |
|------|------|
| 구분 | 공공, 교육 |
| 발주처 | 서울시교육청 |
| 등록자 수 | 6,536명 이상 (2024년 9월 기준) |
| 참여학교 수 | 30개 |
| 이용대상 | 서울 특성화고등학교 학생, 교직원, 방문자 |
| 목적·효과 | 국내 최초 스마트 학생증 및 데이터 기반 활용서비스 / 학생·학부모 서비스 이용 편의성 향상 및 행정 효율성 증대 |

**구현 내용:** 모바일 학생증, 모바일 교직원증, 포트폴리오, 출입관리

**주요 성과:**

- 발행 신분증: 6,536개 (2024.09.18 기준)
- 누적 등록인원: 3.6만명 이상 (기대)
- 2024년까지 모든 서울 특성화고 확산 예정
- 데이터 보유 및 활용기관 4곳 연계 예정 (한국산업인력공단, 대한상공회의소, ETEVERS Learning, 푸른나무재단)

**키워드:** `서울시교육청` `모바일 신분증` `자격증·증명서` `출입관리` `정부기관` `5만명`

### 2. 국회 모바일 신분증

**블록체인 기반 모바일 신원증명 및 간편인증 시스템**

| 항목 | 내용 |
|------|------|
| 구분 | 공공, 입법·행정 |
| 발주처 | 국회사무처 |
| 등록자 수 | 4,900명 이상 (2023년 8월 기준) |
| 참여 소속기관 | 국회사무처, 국회도서관, 국회예산정책처, 국회입법조사처 |
| 이용대상 | 소속 의원, 공무원, 인턴, 공무직 |
| 목적·효과 | 디지털국회 대혁신 세부과제 안전·편리한 원패스 국회 구현 / 공무원증 안정성 및 사용자(국회의원·직원) 편의성 강화 |

**구현 내용:** 모바일 신분증, 모바일 출입증 및 시스템 로그인, 교육출석

**주요 서비스 (6가지):** 모바일 공무원증, 증명서 발급 및 제출, 교육 출결, 도서대출, 정보시스템 로그인, 출입게이트 인증

**키워드:** `국회사무처` `모바일 신분증` `출입관리` `보안로그인` `증명서` `정부기관` `5천명`

### 3. 인천공항 스마트패스 (ICN SMARTPASS)

**여권·탑승권 없이 사전 등록한 얼굴인식 정보로 출국하는 비대면 서비스**

| 항목 | 내용 |
|------|------|
| 구분 | 항공 운항업 |
| 발주처 | 인천국제공항 |
| 가입자 수 | 170만 명 이상 (2024년 2월 기준) |
| 참여 항공사 | 대한항공, 아시아나, 델타항공, 제주항공 등 총 6개 |
| 이용대상 | 인천국제공항을 이용하는 국민, 외국인 여객 |
| 목적·효과 | 출입국 프로세스 간소화로 수속시간 단축 및 만족도 개선 / 국내 최초 공항여객 생체정보 활용 비대면 신원확인 서비스 |

**구현 내용:** 스마트패스 ID, 공항시설 통과, 백드랍(수하물 위탁)

**주요 성과:**

- 스마트패스 ID 발급 (얼굴 등록인원): 150만명+ (2024.02.28 기준)
- 총 가입자 수: 204만 4,527명 (2024.05.23 기준)
- 비행기 탑승 경험: 5.5만회+ (출시 1개월 기준)
- 출국게이트 입장 예약 건수: 100건+ (출시 1개월 기준)

**키워드:** `인천국제공항` `모바일 신분증` `비대면신원확인` `출입관리` `생체인식` `공기업` `5만명`

### 4. 밀리패스 (MILPASS)

**온오프라인 어디서나 군인과 군인가족임을 증명할 수 있는 마이데이터 서비스**

| 항목 | 내용 |
|------|------|
| 구분 | 공공, 국방 |
| 발주처 | 대한민국 육군 |
| 가입자 수 | 35만 명 이상 (2022년 9월 기준) |
| 제휴업체 수 | 500여 개 |
| 이용대상 | 대한민국 육군/해군/해병대 현역 및 예비역, 군인가족 |
| 목적·효과 | 모바일 신분증 및 증명서로 신분확인 절차 간소화 / 국내 최초 현역 데이터 수집 활용 복지 서비스 플랫폼 |

**구현 내용:** 군/군인가족 신분증, 모바일 출입증, 제휴 서비스 연계, 모바일 증명서

**주요 성과:**

- 발행 신분증: 40만명+
- 2030년까지 200만명+ 가입 목표
- 할인 및 복지 혜택서비스 분야: 15가지 (온라인 쇼핑, 여행/숙박, 영화/공연, 버스예매, 자격증/교육, 이사 견적, 전용보험, 금융, 전자책, 음악 스트리밍, 이동통신, 철도예매, 민간바우처, 전통시장, 장기렌터카)

**키워드:** `대한민국 육군` `모바일 신분증` `출입관리` `바우처` `복지몰` `정부기관` `40만명`

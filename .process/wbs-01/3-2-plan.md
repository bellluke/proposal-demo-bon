# 3-2 계획: 출석체크 페이지

## 목표
월간 캘린더 뷰에서 출석 도장을 찍고, 연속 출석 보상을 확인하는 데모.

## 구성 요소

### 1. AttendanceCalendar.tsx 컴포넌트
- 월간 캘린더 그리드 (일~토, 7열)
- 출석한 날: 파충류 발자국 도장 표시 (🦎 이모지 + 초록 배경)
- 오늘: 하이라이트 테두리
- 미래 날짜: 비활성

### 2. 출석체크 버튼
- "출석하기" 버튼 (오늘 미출석 시)
- 출석 시 도장 찍히는 애니메이션 (scale bounce)
- 출석 완료 상태 표시

### 3. 연속 출석 보상 표시
- 보상 마일스톤 카드 4개:
  - 일일: 50P
  - 7일 연속: 500P 보너스
  - 14일 연속: 10% 할인 쿠폰
  - 30일 개근: 무료배송 + 2,000P
- 현재 연속 출석일 수 표시
- 진행률 바

### 4. 상태 관리
- localStorage: 출석 기록 배열 (날짜 문자열)
- 연속 출석일 계산
- 데모 리셋 버튼

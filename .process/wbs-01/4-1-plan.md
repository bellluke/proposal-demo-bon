# 4-1 계획: 상품 태깅 페이지

## 목표
오늘의집 '집들이' 스타일의 피드에서, 사육장 사진 위에 좌표 기반 상품 태그(+)를 배치하고
클릭 시 상품 정보 팝업을 표시하는 데모.

## 구성 요소

### 1. 데이터 구조 (src/data/tagging.ts)

**상품 3개:**
1. 두렙 프리미엄 적재사육장 90x45x45 — 189,000원
2. 두렙 레오파드 게코 은신처 M — 15,900원
3. 두렙 UVB 바스킹 라이트 50W — 23,000원

**피드 게시물 3개:** (각각 1장의 이미지 + 상품 태그 2~3개)
- 게시물 1: "나의 첫 레오파드 게코 사육장" — 사육장 + 은신처 태그
- 게시물 2: "블루 이구아나 바스킹 셋업" — 사육장 + 바스킹 라이트 태그
- 게시물 3: "크레스티드 게코의 아늑한 집" — 사육장 + 은신처 + 라이트 태그

각 태그는 `{ productId, x: number, y: number }` (퍼센트 좌표)

### 2. 이미지
- placeholder 색상 배경 + 텍스트 (실제 사육장 이미지 없으므로)
- CSS gradient 배경으로 자연 느낌 연출

### 3. ProductTaggingFeed.tsx 컴포넌트
- 피드 형식 (세로 스크롤)
- 각 게시물: 유저 아바타/닉네임, 이미지, 설명, 좋아요/댓글
- 이미지 위 태그 마커 (+ 아이콘, absolute position)
- 마커 클릭 시 팝업 (상품명, 가격, 썸네일)
- 이미지 클릭으로 전체 태그 토글 표시

### 4. 상품 정보 팝업
- 마커 근처에 말풍선 형태
- 상품 썸네일(컬러 블록) + 상품명 + 가격
- "상품 보기" 버튼 (데모이므로 # 링크)

### 5. 반응형
- 모바일: 전체 너비, 태그 위치 퍼센트 기반이므로 자동 스케일
- 데스크탑: 최대 너비 제한

## 기술 구현
- "use client" (클릭 인터랙션)
- 좌표: percentage 기반 absolute positioning
- 팝업: state로 현재 활성 태그 관리
- 이미지: CSS gradient placeholder

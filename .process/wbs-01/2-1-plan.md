# 2-1 계획: 데모 홈 페이지

## 목표
프로젝트 소개와 각 데모 기능으로의 네비게이션을 제공하는 메인 랜딩 페이지 구현.

## 구성 요소

### 1. 히어로 섹션
- "두렙(DooRep)" 브랜드 타이틀
- "파충류 전문 커뮤니티 통합 커머스 플랫폼" 서브 타이틀
- 핵심 가치 3가지 (콘텐츠→구매, 재방문, 브랜드 정체성)
- CTA: "데모 살펴보기" 버튼

### 2. 핵심 기능 카드 섹션
3개의 기능 카드 (각각 데모 페이지로 링크):
1. **상품 태깅** - 사육장 사진 + 상품 태깅 설명, /product-tagging 링크
2. **게이미피케이션** - 룰렛 + 출석체크 소개, /roulette, /attendance 링크
3. **커뮤니티 커머스** - 플랫폼 전체 비전 설명

### 3. 데모 네비게이션 그리드
- 4개 카드: 스타일 가이드, 룰렛, 출석체크, 상품 태깅
- 각 카드에 아이콘 + 설명 + 링크

### 4. 제안서 핵심 정보 섹션
- 프로젝트 개요 (2단계 런칭 전략)
- 기대 효과 카드

## 기술 구현
- 서버 컴포넌트 (정적 페이지)
- Lucide 아이콘 활용
- Card, Badge 컴포넌트 재사용
- 반응형 그리드 (1열 → 2열 → 3열)

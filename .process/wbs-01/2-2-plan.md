# 2-2 계획: 스타일 가이드 페이지

## 목표
디자인 시스템의 컬러, 타이포그래피, UI 컴포넌트를 시각적으로 보여주는 페이지.

## 구성 요소

### 1. 컬러 팔레트 섹션
- 각 색상을 사각형 스와치로 표시
- 색상명 + HEX 코드 + 용도 설명
- 그룹: Primary, Secondary, Accent, Background, Text, Status

### 2. 타이포그래피 섹션
- 폰트 패밀리 (Pretendard / Inter)
- 크기 스케일 (12px ~ 48px) 실제 렌더링

### 3. 버튼 컴포넌트 쇼케이스
- 4종 변형 × 3종 크기 조합 표시

### 4. 카드 & 배지 쇼케이스
- Card: 기본/호버
- Badge: 6종 변형

### 5. 간격 & 라운딩 가이드
- border-radius 예시
- 그림자 레벨 예시

## 기술 구현
- 서버 컴포넌트 (정적)
- Button, Card, Badge 컴포넌트 직접 렌더링
- 색상 데이터는 배열로 정의

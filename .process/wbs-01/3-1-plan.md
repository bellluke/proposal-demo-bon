# 3-1 계획: 룰렛 페이지

## 목표
파충류 테마 룰렛 휠을 구현하여 매일 포인트/쿠폰을 획득하는 게이미피케이션 데모.

## 구성 요소

### 1. 룰렛 휠 컴포넌트 (RouletteWheel.tsx)
- CSS transform: rotate() 기반 회전 애니메이션
- 7개 보상 슬라이스 (각각 다른 색상)
- 중앙 "SPIN" 버튼
- 상단 화살표 마커
- 회전 시 ease-out 감속 효과

### 2. 보상 항목 (plan.md 데이터)
| 보상 | 확률 |
|------|------|
| 100 포인트 | 30% |
| 200 포인트 | 25% |
| 500 포인트 | 15% |
| 1,000 포인트 | 10% |
| 사료 샘플 교환권 | 10% |
| 5% 할인 쿠폰 | 8% |
| 무료배송 쿠폰 | 2% |

### 3. 당첨 결과 모달
- 보상 아이콘 + 축하 메시지
- "확인" 버튼으로 닫기

### 4. 상태 관리
- `hasSpunToday`: localStorage 기반 일일 1회 제한
- 데모용 "다시 돌리기" 리셋 버튼
- 당첨 결과 표시

### 5. 보상 목록 사이드 패널
- 전체 보상 + 확률 표

## 기술 구현
- "use client" (인터랙티브)
- CSS transition + transform: rotate() for 회전
- 가중치 기반 랜덤 선택 알고리즘
- localStorage로 참여 기록

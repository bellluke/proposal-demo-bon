# 1-1. 슬라이드 데이터 구조화

## 목표
`docs/proposal.md` 내용을 10장의 슬라이드 데이터로 구조화

## 산출물
- `src/data/slides.ts` — 슬라이드 타입 정의 + 데이터

## 슬라이드 타입
- `cover`: 표지 (제목, 부제, 메타 정보)
- `bullets`: 불릿 리스트 (제목 + 항목들)
- `table`: 표 형식 (제목 + 행/열 데이터)
- `split`: 좌우 분할 (왼쪽 텍스트 + 오른쪽 리스트)
- `timeline`: 타임라인 (단계별 진행)
- `closing`: 마무리 (Q&A, 연락처)

## 주의사항
- "두렙(DooRep)" 브랜드명 제거 (이전 수정사항 반영)
- "(B.O.N)" 영문명 제거
- 실제 제안서 내용 충실 반영
